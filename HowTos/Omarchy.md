# Omarchy Window Controls
## Window Movement
| Keybinding | Action |
|------------|--------|
| SUPER + Arrow keys | Focus window in direction |
| SUPER SHIFT + Arrow keys | Swap window in direction |
| SUPER + LEFT MOUSE BUTTON | Drag/move window |
| SUPER + RIGHT MOUSE BUTTON | Resize window |
## Window Actions
| Keybinding | Action |
|------------|--------|
| SUPER + W | Close window |
| CTRL ALT + DELETE | Close all windows |
| SUPER + T | Toggle floating/tiling |
| SUPER + O | Pop window out (float & pin) |
| SUPER + P | Pseudo window (split) |
| SUPER + ALT + S | Move to scratchpad |
| SUPER + BACKSPACE | Toggle window transparency |
| SUPER CTRL + BACKSPACE | Toggle single-window square aspect |
| SUPER SHIFT + BACKSPACE | Toggle window gaps |
| SUPER + J | Toggle window split |
## Workspace
| Keybinding | Action |
|------------|--------|
| SUPER SHIFT + 1-9 | Move window to workspace 1-9 |
| SUPER SHIFT + 0 | Move window to workspace 10 |
| SUPER SHIFT ALT + 1-9 | Move silently to workspace 1-9 |
| SUPER SHIFT ALT + 0 | Move silently to workspace 10 |
## Grouping
| Keybinding | Action |
|------------|--------|
| SUPER + G | Toggle window grouping |
| SUPER + ALT + TAB | Next window in group |
| SUPER SHIFT ALT + TAB | Previous window in group |
| SUPER ALT + 1-5 | Switch to group window 1-5 |
| SUPER ALT + G | Move active window out of group |
| SUPER ALT + Arrow | Move window to group (left/right/top/bottom) |
| SUPER CTRL + LEFT/RIGHT | Move grouped window focus |
## Resize Mode (SUPER + R)
| Keybinding | Action |
|------------|--------|
| Arrow keys | Resize in direction |
| SUPER + minus | Expand window left |
| SUPER + equal | Shrink window left |
| SUPER SHIFT + minus | Shrink window up |
| SUPER SHIFT + equal | Expand window down |
## Window Cycle
| Keybinding | Action |
|------------|--------|
| ALT + TAB | Cycle to next window |
| SHIFT ALT + TAB | Cycle to prev window |
